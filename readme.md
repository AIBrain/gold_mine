Keyboard Shortcuts
==================

- `ctrl + n` -- Start a `n`ew game.
- `ctrl + s` -- Open the `s`tatistics window.
- `ctrl + f` -- Try to move all the possible cards from the waste/tableau to the `f`oundation.
- `ctrl + a` -- Open the `a`bout webpage.
